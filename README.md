# Blood Pressure Classification

This project analyzes the Framingham Heart Study dataset to predict blood pressure disorders.

Blood pressure status is defined using WHO systolic and diastolic thresholds, and classified as normal or disorder.

## Methods
- Data cleaning and preprocessing
- Target variable creation (BP disorder)
- Oversampling for class imbalance
- Decision Tree, Random Forest, XGBoost models
- Feature importance analysis

## Tools
- Python
- Scikit-learn
- XGBoost
- Pandas, NumPy
